# Arcadia Sans

A modern hybrid geometric font with angled corners and variable weight.

![](./Specimen.png)

## Features

* Variable weight
* Designed for both body text and headers
* Tabular numbers (always-on): the 10 digits are equal width, as are period, comma, colon, and semicolon
